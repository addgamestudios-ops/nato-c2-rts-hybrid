## TECHNICAL REQUIREMENTS

### Core Architecture
- **Unity 2022.3+** (target 2022.3.20f1 or later)
- **Nebukam ORCA** (com.nebukam.orca) for high-performance local avoidance (Job System + Burst)
- **HPA*** (Hierarchical Pathfinding A*) with Catmull-Rom smoothing for global paths
- **Milsymbol** library integration (via WebGL bridge + C# wrapper) for MIL-STD-2525E / APP-6E symbols
- **Multi-layer altitude system** (Ground / Low / High) with layer-aware neighbor queries

### Key Systems to Implement

1. **NATO_C2_Manager.cs** — Central orchestrator
   - Manages all agents, dynamic obstacles, pathfinding, and AI
   - Handles layer registration and cross-layer interaction rules
   - Provides public API for selection, commands, and autonomous mode

2. **Agent.cs** — Unit wrapper
   - Supports drone, tank, UGV, helicopter, etc.
   - Stores preferredVelocity, current velocity, radius, maxSpeed, unitType
   - Visual feedback (color, symbol, health bar)

3. **ORCA.cs** (Nebukam bridge + custom fallback)
   - High-performance reciprocal collision avoidance
   - Supports dynamic obstacles with stronger repulsion
   - Layer-aware neighbor filtering

4. **HPAStar.cs**
   - Hierarchical Pathfinding A* with clustering
   - Catmull-Rom path smoothing
   - LOS optimization (line-of-sight shortcutting)

5. **FormationController.cs**
   - Wedge, Line, Circle formations with live preview
   - Dynamic slot assignment based on unit count and type
   - Maintains formation under ORCA influence

6. **AIAutonomousMode.cs** (Mythos AI)
   - Threat prediction with 8–12 second lookahead
   - Toggleable autonomous decision making
   - Opportunity detection and evasion vectors

7. **MilsymbolBridge.cs**
   - C# ↔ JavaScript bridge for WebGL
   - Symbol caching
   - Fallback placeholder system for Editor
   - Supports APP-6E and MIL-STD-2525E SIDC codes

8. **TacticalHUD.cs + CommandRadialMenu.cs**
   - Drag-box selection
   - Radial command wheel (MOVE, ATTACK, LOITER, SWARM, RTB, HOLD)
   - Autonomous Mode toggle button
   - Threat visualization overlay

9. **DynamicObstacle.cs + DynamicObstacleSpawner.cs**
   - Moving and static obstacles
   - Runtime spawning for demo purposes
